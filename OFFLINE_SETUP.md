# Offline Setup Guide

## What to Download

### 1. GSAP Library (for animations)
Download from: https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js
- Save as: `lib/gsap.min.js`

Download from: https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/ScrollTrigger.min.js
- Save as: `lib/ScrollTrigger.min.js`

### 2. Google Fonts (Optional - Already using system fallbacks)
If you want the exact Inter & JetBrains Mono fonts offline:
- Download from: https://fonts.google.com/download?family=Inter
- Download from: https://fonts.google.com/download?family=JetBrains%2BMono
- Extract and place in the `fonts/` folder
- Update the CSS @font-face rules

## Steps to Download Files

**Easiest Method:**
1. Open each link above in your browser
2. Right-click → "Save file as"
3. Save in the corresponding `lib/` or `fonts/` folder
4. Make sure filenames match exactly

## Result
Once files are downloaded, your portfolio will work 100% offline!
